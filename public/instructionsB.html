Anleitung

Falls Ihr Kind ein Wort schon einmal gesagt hat, wählen Sie JA. Dafür können Sie entweder auf das grüne Häkchen drücken oder die Karte nach rechts wischen. Um NEIN zu wählen, drücken Sie auf das rote Kreuz oder wischen sie die Karte nach links. Fall Sie an einem Computer sind, können Sie auch die Pfeiltasten verwenden (rechts = ja; links = nein).

Falls Ihr Kind ein Wort anders ausspricht oder eine andere Form benutzt (z.B. "Raffe" statt Giraffe oder "Karotte" statt Möhre), wählen Sie trotzdem JA.

Sie können jederzeit eine Pause einlegen und später weitermachen. Falls Sie das Fenster schließen, klicken Sie einfach wieder auf den Studienlink aus der Email.

Nun kann es losgehen – Vielen Dank und viel Spaß!
